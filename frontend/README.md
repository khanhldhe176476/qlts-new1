# Frontend - Quản lý tài sản với Ant Design

Frontend React với Ant Design cho hệ thống quản lý tài sản.

## Cài đặt

```bash
cd frontend
npm install
```

## Chạy development server

```bash
npm run dev
```

## Build cho production

```bash
npm run build
```

Sau khi build, files sẽ được tạo trong `../static/frontend` và Flask sẽ serve chúng.
